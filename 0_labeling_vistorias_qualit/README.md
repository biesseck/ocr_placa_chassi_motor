## Prerequisites
- Git: https://github.com/git-for-windows/git/releases/download/v2.52.0.windows.1/Git-2.52.0-64-bit.exe
- Python 3.9: https://www.python.org/ftp/python/3.9.9/python-3.9.9-amd64.exe


## Download and installation
```
git clone https://github.com/biesseck/ocr_placa_chassi_motor.git
cd ocr_placa_chassi_motor/0_labeling_vistorias_qualit
pip install -r requirements.txt
./install.bat    # it created a shortcut at user's Desktop
```


## Running
```
.\main_labeling_vistorias_qualit.bat
```
